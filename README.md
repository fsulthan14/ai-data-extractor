# 🚀 AI Data Extractor (IMG to Structured Data)

Aplikasi berbasis Python dan Streamlit yang memanfaatkan kekuatan **Gemini API (Google AI Studio)** untuk mengekstrak data dari dokumen gambar (JPG, PNG) atau PDF (seperti mutasi rekening, nota, atau dokumen lainnya) menjadi format terstruktur (CSV/JSON).

## ✨ Fitur Utama
- **Batch Processing:** Upload banyak file sekaligus untuk diekstrak.
- **Multimodal AI:** Menggunakan Gemini 3.0 Flash Preview untuk pembacaan OCR yang akurat.
- **Ekspor Data:** Hasil ekstraksi bisa langsung diunduh ke format CSV atau JSON.
- **Custom Prompt:** Atur instruksi ekstraksi sesuai kebutuhan dokumen.
